# Interpretación comparativa

Se observa una tendencia clara:

* **Identidad ~99% → RMSD < 1 Å**
  (estructuras prácticamente indistinguibles)

* **Identidad ~97% → RMSD ~1.8 Å**
  (misma estructura con pequeñas diferencias locales)

* **Identidad ~37% → RMSD ~3 Å**
  (misma arquitectura global, pero mayor divergencia en loops y orientación relativa de elementos secundarios)

Esto confirma la relación esperada:

> A mayor identidad de secuencia, menor RMSD y mayor conservación estructural.
